# Phase 3 — Policy Decision and Tool-Call Firewall

## Goal

Demonstrate allow, deny, approval-required, and redact decisions for governed AI workflow requests.

## Checklist

- [x] Classification-aware policy evaluator
- [x] Tool-call blocking rules
- [x] Human approval gate result
- [x] OPA/Rego-style reference policy

## Boundary

The lab evaluator demonstrates governance behavior. It does not replace enterprise IAM, OPA, DLP, SIEM, PAM, or clinical/financial authorization controls.
